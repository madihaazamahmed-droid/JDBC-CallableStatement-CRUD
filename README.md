# JDBC CallableStatement CRUD

A Java JDBC project demonstrating the use of `CallableStatement` to perform database operations with MySQL. The project covers database connectivity, table creation, and inserting employee records.

## Features

- JDBC Connectivity
- MySQL Integration
- CallableStatement
- Employee Record Insertion
- Maven Project

## Technologies Used

- Java
- JDBC
- MySQL
- Maven
- Eclipse IDE

## Project Structure

```
src
└── storedprocedure
    └── Sample.java
```

## Database

Database Name:

```
employee2
```

Table:

```
emp_det
```

Columns

- e_id
- e_name

## Concepts Covered

- Loading MySQL Driver
- Database Connection
- Statement
- CallableStatement
- SQL Execution
- User Input using Scanner

## How to Run

1. Clone the repository.
2. Create the MySQL database.
3. Update the database credentials.
4. Run the Java application.
5. Verify inserted records in MySQL.
